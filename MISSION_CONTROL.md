# Mission Control

A virtual office dashboard to view agents, status, and chat.

## Dev

```bash
cd mission-control
npm install
npm run dev
```

## Deploy

Connect this repo to Vercel. No secrets in git.

## Roadmap

- Live agent feeds (OpenClaw session status, job queue)
- Realtime chat (Socket.IO + Vercel serverless / Edge)
- GitHub widgets (PRs, actions, commits)
- Vercel widgets (deployments, logs)
